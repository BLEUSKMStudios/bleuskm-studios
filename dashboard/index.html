Setup Guide
Access · Voice JARVIS · Daily Text · GitHub · Higgsfield · Granola
1. You're Already Here — Access This Dashboard Anywhere
This file lives at bleuskm.com/dashboard/index.html
Bookmark it on your phone and laptop. Nobody else sees it unless you share the link.

To password protect it: go to Netlify → your site → Site Settings → Access Control → add a password. Free.
2. Voice JARVIS — Say "Hey JARVIS"
git clone https://github.com/ethanplusai/jarvis cd jarvis pip install -r requirements.txt Create a .env file and add:
ANTHROPIC_API_KEY=your_key_here Then run: python main.py Say "Hey JARVIS" → it wakes up → you speak → it responds in voice.

Tell me when you're set up and I'll write your full custom system prompt with all Bleuskm context baked in.
3. Daily Text from JARVIS (Morning + Night)
1. Sign up at twilio.com (free trial) → get a phone number
2. Sign up at zapier.com (free)
3. Create Zap:
   Trigger: Schedule → Every day 8:15 AM
   Action: Claude API → morning briefing prompt
   Action: Twilio SMS → your phone number
4. Repeat for 7PM night summary

Send me your Twilio number and I'll write both prompts for you to paste into Zapier.
4. Updating This Dashboard via GitHub
Any time you want to update the dashboard — tell JARVIS what to change, it writes the code, then run:
git add dashboard/index.html git commit -m "update dashboard" git push origin main Netlify auto-deploys in ~30 seconds. Dashboard updates live.
5. Higgsfield AI Video
Go to higgsfield.ai → create account → use the AI Tools tab here to generate an optimized prompt → paste into Higgsfield → download → use in your content.

Best for: atmospheric mood clips for @Bleuskm.studios, scene visualization before production, B-roll for The Final Hand social posts.
6. Granola — Automatic Meeting Notes
Download granola.so (Mac app) → it runs in the background and records + transcribes your calls automatically.

After any call: Granola gives you a clean summary → paste it into the AI Tools tab → JARVIS extracts action items, decisions, and deadlines.

Use for: client calls, crew meetings, casting conversations, any creative call you want notes from.
